# سامانه ثبت کارآموزی دانشجویان

نام دانشجو: فاطمه مولایی

شماره دانشجویی: 0312117475021

# توضیح کلی پروژه
این پروژه با هدف طراحی یک سامانه هوشمند ثبت کاراموزی دانشجویان طراحی میشود تا دانشجویان به راحتی و در کمترین زمان بتوانند اطلاعات محل کار و ساعات و گزارش هارا وارد کرده تا استاد راهنما تایید نهایی کند.

# اهداف پروژه

هدف 1 :طراحی سامانه هوشمند

هدف 2 :ثبت اطلاعات کاراموزی در کمترین زمان و دسترسی آسان.

هدف 3 : ارتباطی سریع با استاد راهنما.
